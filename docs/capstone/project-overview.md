## Key Learning Outcomes Applied
* **ROS 2:** Using the framework for communication and control.
* **Gazebo/Isaac Sim:** Physics-accurate simulation environment.
* **Isaac ROS:** Hardware-accelerated perception and navigation (VSLAM).
* **VLA/LLMs:** Translating natural language into executable robot actions.
